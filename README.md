# Week of Maths

Mathematical quizzes for Mathematics Week

## Build

Install the depencies
```
npm install
```

Lint the JS
```
npm run lint
```

Build the CSS + JS
```
npm run build
```

Build the CSS
```
npm run build-css
```

Build the JS
```
npm run build-js
```

Watch CSS + JS for development
```
npm run watch
```
